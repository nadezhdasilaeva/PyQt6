Добро пожаловать в desktop приложение для симуляции терминала охранника посещаемости сотрудников организации! Это руководство пользователя поможет вам ознакомиться с основными функциями и возможностями приложения.

1. Установка ПО
- Установить приложение PyCharm c официального сайта JETBRAINS
- Перейдите по ссылке GitHub и скачать файлы terminal.py, passage.py, base_data.db, base_data_2.db
- Для посмотра информации в базе данных, также можете установить DBeaver

2. Вход в приложение
- Все скаченные файлы перенесите в папку проекта
- Запустите файл terminal.py

3. Основной интерфейс
- После запуска, вам будет доступно окно с возможность введения данных о человеке.
- Наличие кнопки Войти.
- Наличие кнопки Выйти.
- Наличие кнопки Временный пропуск.

4. Учет посещаемости
- В полях Фамилия, Имя, Отчество вводим соотвестсвующего человека.
- Если человек, хочет войти нажмите кнопку Войти.
- Если человек, хочет выйти нажмите кнопку Выйти.
- Если человек не был найден в базе данных, то нажав кнопку Временный пропуск и введя необходимые данные выдать пропуск нажав кнопку Выдать пропус.

5. Выход из приложения
- Чтобы выйти из приложения нажмите на крестик в правом верхнем углу окна.

Это основное руководство пользователя для desktop приложения по симуляции терминала охранника посещаемости сотрудников организации. Желаем вам успешной и удобной работы с приложением!
